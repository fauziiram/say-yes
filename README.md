# Pencet Iya! 💖

Aplikasi prank pertanyaan interaktif yang lucu dan gemas. Buat pertanyaan jebakan, kirim link ke pacar/gebetan/teman, lalu biarkan mereka frustasi karena tombol **"Tidak"** selalu kabur dari kursor! 😂

## Fitur

- ❤️ **Buat Prank Kustom** — Tulis pertanyaan, pilih animasi perayaan, dan atur pesan sukses
- 🏃 **Tombol "Tidak" Kabur** — Tombol menolak selalu lari, muter, mengecil saat didekati kursor
- 🎉 **Confetti & Animasi** — Perayaan spesial saat akhirnya menekan "Iya"
- 🔗 **Link Shareable** — Generate link dan kirim langsung via WhatsApp, Telegram, atau copy ke clipboard
- 📱 **Responsive** — Tampilan optimal di desktop maupun mobile

## Tech Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS 4
- Motion (Framer Motion)
- Lucide React Icons

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```
2. Jalankan dev server:
   ```bash
   npm run dev
   ```
3. Buka `http://localhost:3000` di browser

## Build

```bash
npm run build
npm run preview
```

## Type Check
